# seanmontague.com — Design System

A personal site design system built on Drupal with Storybook for component development.
Follows the [Surface theme](https://github.com/uclahs/surface) pattern hierarchy.

## Patterns organization

* **Base** — Global CSS, design tokens, utilities, typography, and reset styles.
* **Elements** — The smallest indivisible patterns: button, byline, date, eyebrow, images, links, list, pager, quote, readtime, skip-link, text, title, video.
* **Components** — Composed from two or more elements: nav, hero, marquee, map, card, places-card, about, contact, footer, elevation-profile, stats-bar.
* **Collections** — Composed from two or more components: article, article-header, map-section, article-map-section.
* **Layouts** — Drupal region and page-level wrappers: block, region, site-container, site-header, site-footer, site-navigation, site-homepage.
* **Pages** — Full page prototypes: page-homepage, page-article.
* **Theme** — Drupal-specific admin UI overrides only (ckeditor, toolbar, media-library, etc.).

## Design language

**Palette** — Warm off-white (`--bg: #f7f6f2`), near-black type (`--bright: #161410`),
forest green (`--forest: #3a5a40`), trail brown (`--trail: #7a3410`), sky blue (`--sky: #4a7c9e`).

**Typography** — Three-font stack defined in `props/nek.css`:
- `Bebas Neue` (`--font-display-nek`) — hero titles, stats values
- `Cormorant Garamond` (`--font-serif-nek`) — body text, article headings, italic accents
- `DM Mono` (`--font-mono-nek`) — labels, navigation, metadata, category tags

## Tooling

* [Vite](https://vite.dev/) (JS Edition)
* [Storybook](https://storybook.js.org/) (HTML Edition)
* Twig + PostCSS + Vanilla JS
* Leaflet.js for interactive maps
