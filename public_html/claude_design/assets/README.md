# Assets

## `icons/`
Hand-drawn SVG icons copied verbatim from `public/seanmontague.html`. 1.5px stroke, no fill, matches Lucide's visual language. Use `currentColor` so they inherit text color.

- `globe.svg` — Kingdom Trails category
- `mountain.svg` — Burke Mountain category
- `tree.svg` — Permaculture category
- `map.svg` — Leaflet Maps category
- `pencil.svg` — Writing category
- `arrow-right.svg` — generic forward arrow

For icons not in this set, use [Lucide](https://lucide.dev) at 1.5px stroke weight as a drop-in match.

## `logo/`
Surface has no graphic logo. The wordmark is **mono text**: `Sean Montague / Burke, VT`. See `logo/wordmark.html` for the canonical treatment.
