# Sean Montague · Editorial Site — UI Kit

A pixel-accurate React (JSX + Babel inline) recreation of the editorial site at `public/seanmontague.html` and `public/seanmontague-article.html`.

## Files
- `index.html` — interactive demo: home view + article view, click-through navigation
- `Nav.jsx` — fixed nav with wordmark
- `Hero.jsx` — hero with Leaflet map background, categories
- `Marquee.jsx` — mono scrolling ticker
- `MapSection.jsx` — two-col featured map
- `ArticleCard.jsx`, `ArticleGrid.jsx` — 3-up bordered card grid
- `PlaceCard.jsx`, `PlaceGrid.jsx` — 2-up numbered place cards
- `AboutSection.jsx` — about block with skills list
- `ContactSection.jsx` — dark contact CTA
- `Footer.jsx` — dark footer
- `StatsBar.jsx` — 5-up editorial stats
- `Blockquote.jsx` — pull-quote
- `ArticleView.jsx` — full article view composition

All components use `--forest` hover rail, zero rounding, warm paper ground.
